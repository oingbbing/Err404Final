# 파이널 프로젝트
🍔 Eatting – 배달 주문 웹 서비스

**Eatting(잇팅)**은 배달의 민족, 요기요와 같은 배달 플랫폼을 참고하여
고객 · 가게 · 라이더 간 실시간 주문 흐름을 구현한 웹 기반 배달 서비스입니다.

보다 편리하고 직관적인 주문 경험에 대한 요구가 증가하고 있습니다.
